# Changelog

## [1.0.2] - 2026-03-12
### Added
- Опция `button.useDefaultIcon` и тест на дефолтную иконку.
- Публичные примеры для `uiMount` и отключения UI-стилей в демо.
- Расширенная документация: полный список опций, примеры, пояснения по `labels`.

### Changed
- `text-align` и `font-family` применяются только при явном выборе параметра.
- Обновлена демо-страница: встроенный слот для кнопки, переключатель UI-стилей, локальные изображения.

### Fixed
- Исправлена подпись для `serif` в панели ("С засечками").

## [1.0.0] - 2026-03-09
### Added
- Базовый API `createA11yView` для `auto` и `control` режимов.
- Поддержка CJS/ESM и типов TypeScript.
- Панель управления с локалями `ru/en`.
- Хранение состояния: `none | localStorage | sessionStorage`.
- Настройки текста, контраста, ссылок, изображений, анимаций.
- Отдельные стили пакета (`core.css`, `ui.css`, `index.css`).
- Demo-страница для проверки сценариев.
- Тесты на ключевые сценарии (`vitest`).

### Changed
- Рефакторинг `index.ts` с выносом логики в `src/core/*`.

### Fixed
- Добавлены базовые стили `colorScheme` (`light`, `dark`, `sepia`) в `core.css`.
- Исправлены проблемы с открытием/закрытием панели и синхронизацией UI-состояния.
