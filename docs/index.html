A11y View Mode - Demo

Демо покрывает все публичные настройки: режимы, текст, контраст, темы, изображения, хранение, язык, стили и варианты кнопки.

1. Режим и инициализация

AUTO: только кнопка. По нажатию применяется пресет. CONTROL: кнопка + панель с пользовательскими настройками. Destroy: удаляет кнопку/панель и очищает атрибуты режима.

6. Изображения и strategy

Важно: стратегии transparent и collapse работают только когда режим изображений = hide. transparent: визуально скрывает контент, старается мягко сохранить поток. collapse: сильнее сжимает блоки с изображениями, уменьшая их влияние на layout.

7. Хранение состояния

NONE: состояние не сохраняется. LOCAL: сохранение в localStorage до ручной очистки. SESSION: сохранение в sessionStorage до закрытия вкладки.

8. Язык панели

Переключение языка на лету работает в control-режиме через публичный API setLocale.

9. Варианты кнопки

UI mount slot

Проверка режимов кнопки: только label, только icon, icon + label. Укажите путь до иконки в настройках кнопки, используйте флаг useDefaultIcon для включения/выключения дефолтной svg. Встраивание в разметку: используйте uiMount + disableAutoPosition.
